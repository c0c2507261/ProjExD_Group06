# CosmoSomething

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

* 宇宙空間で戦闘機が敵を打つシューティングゲーム

## ゲームの遊び方
* 矢印キーで戦闘機を操作し，スペースキーでビームを発射し敵を撃破する。
* xキーで貫通ビームを発射できる。sキーを押すことで3回までシールドを生成し、シールド1個で1回まで被弾を防げる。
* GameOverの際にEnterキーを押すとコンテニューできる。

## ゲームの実装

### 共通基本機能
* 背景画像とプレイヤー(戦闘機)と敵の描画
* Scoreの表示
